# Nix Package Manager
**Nix Package Manager** — менеджер пакетов, ориентированный на воспроизводимость сборок.  
- Проверяет корректность входных для окружения
- Является декларативным* (а не императивным*).

---
\*Декларативный подход означает, что вы настраиваете конфигурацию компьютера с помощью конфигурационного файла или набора конфигурационных файлов. Если в конфигурационные файлы добавляется что‑либо, это будет добавлено в систему. Если из конфигурационных файлов что‑либо удаляется, это будет удалено из системы. Сохраняя конфигурационные файлы, вы легко можете вернуть состояние системы.

\*Императивный подход означает, что вы настраиваете конфигурацию компьютера через последовательность шагов. Эта последовательность может включать поочерёдную установку необходимых пакетов, изменение настроек в менеджере настроек и/или ручное редактирование системных файлов в каталоге `/etc`.
